## Troubleshooting

**Port 3000 in use?**
```bash
# Kill what's using it
sudo lsof -ti:3000 | xargs kill -9

# Or change port in vite.config.js
```

**Not opening automatically?**
Open http://localhost:3000 manually in your browser.
